# Song Tunes
### About project:
This is an application that simulates a music streaming service, like Spotify. Being able to search for songs, favorite them, etc.
### How to run:
- Clone the repository.
- Run the following commands:
```
npm start

```
or visit:

https://song-tunes.vercel.app/
